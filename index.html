공학도서 상품 자동 업로드 · 관리자 전용

1. 엑셀 파일 선택





· 파일 이름에서 발행사를 자동으로 추정합니다. 필요하면 오른쪽에서 수동으로 변경할 수 있습니다.
· 이 화면은 “업로드 전용 관리자 도구” 이며, 공식 홈페이지 화면과 분리되어 동작합니다.

2. 열 매핑 설정



해외 메타데이터 열 매핑 공식 업로드 엑셀 열
· 이 표의 선택 상태는 발행사별로 자동 저장할 수 있습니다. (build-up 확장 영역)
· 지금은 예시 열 이름으로 테스트할 수 있고, 나중에 엑셀 실제 헤더를 읽어오도록 바꿀 수 있습니다.

3. 업로드 실행 & 로그

· 테스트 기준: (1) 화면 표시 확인 → (2) 업로드 실행 1회 → (3) 오류 0건 확인 시 테스트 완료.
